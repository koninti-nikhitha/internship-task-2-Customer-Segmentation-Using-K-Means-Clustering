# Task 02 – Customer Segmentation Using K-Means Clustering

## 📌 Overview

This project implements the **K-Means Clustering** algorithm to segment retail store customers based on their purchase history. Customer segmentation helps businesses understand customer behavior, identify similar purchasing patterns, and develop targeted marketing strategies.

## 🎯 Objective

To group customers into meaningful clusters using the K-Means algorithm and analyze their purchasing behavior.

## ✨ Features

* Data preprocessing and cleaning
* Feature scaling using StandardScaler
* Customer segmentation using K-Means Clustering
* Optimal cluster selection using the Elbow Method
* Cluster visualization using scatter plots
* Customer behavior analysis

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn
* Jupyter Notebook

## 📂 Dataset

The dataset contains customer purchase information, including features such as:

* Customer ID
* Annual Income
* Spending Score
* Purchase History (if available)

## 📊 Workflow

1. Load the dataset.
2. Perform data preprocessing.
3. Scale the selected features.
4. Apply the Elbow Method to determine the optimal number of clusters.
5. Train the K-Means model.
6. Visualize the resulting customer segments.
7. Analyze the characteristics of each cluster.

## 📈 Results

The model successfully grouped customers into distinct clusters based on purchasing behavior, providing valuable insights for customer profiling and targeted marketing.
